# Lawyer Candidates

Shortlist of compliance counsel candidates. 3 conversations this week per brief Section 7.

## Per-candidate file format
Create one file per candidate: `{firm-slug}.md`. Include:

- Firm name, lead attorney, jurisdictions they practice in
- Fintech / commercial-lending / state-licensing track record
- Hourly rate + flat-fee options for privacy policy + ToS
- Turnaround on typical fintech startup engagement
- Reference clients (anonymized is fine)
- Red flags (conflicts, cap-on-hours, rigidity on scope)
- Call notes with date

## Evaluation grid
Score each candidate 1–5 on:

1. Commercial-lending-broker licensing experience (CA/NY/VA/UT specifically)
2. UDAAP / marketing-copy review comfort
3. Speed (weeks-to-draft on ToS + privacy policy)
4. Price transparency (flat-fee willingness)
5. Fit (do they return calls in hours or days? respect short-form briefs?)

## Decision target
Engage one by **+2 weeks from 2026-04-21** (brief Section 11 horizon).
